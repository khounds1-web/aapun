# Spec Heartbeat

Heartbeat is data, not prose.

```json
{
  "iteration": 1,
  "all_green": false,
  "dimensions": {
    "H1_acceptance_criteria": {
      "status": "RED",
      "evidence": [],
      "reason": "implementation not present"
    }
  }
}
```

If a dimension is GREEN, evidence must be non-empty.
