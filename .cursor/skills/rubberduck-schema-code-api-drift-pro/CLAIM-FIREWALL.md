# Claim Firewall

Block or caveat claims that exceed evidence.

## Block

```text
no drift found
schema is clean
API and DB agree
concept fully mapped
runtime failure predicted
round trip safe
```

unless required evidence exists.

## Require caveat

```text
partial layer inventory
LOW concept fusion
source-only contract
unsupported schema format
dynamic runtime serialization
generated clients not loaded
```

## Final report rule

Each final drift finding must have:

```text
concept
layers compared
dimension
evidence
impact
confidence
falsification recipe
```
