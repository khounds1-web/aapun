# Claim Firewall

Block or caveat claims that:

```text
claim exhaustive without full candidate-generator coverage
claim all functions fingerprinted when prefiltering was used
include skipped/unavailable dimensions in composite score
omit inline implementation search
omit comment-contract mining
omit change-impact preflight before unification plan
omit cluster role classification
have inconsistent cluster member counts
omit falsification recipes
```

Allowed phrasing:

```text
exhaustive for the listed generators
seeded hunt, not unconstrained census
computed across F1/F2/F3/F8; F5/F6 unavailable
strong match across computed dimensions
candidate generation did not cover dynamic runtime-only code
```
