# Claim Firewall

Block:

```text
equivalent
safe to replace
drop-in replacement
no behavior change
all paths matched
tests prove equivalence
```

unless the evidence pack supports the claim under the stated contract.

Required caveats:

```text
unsupported language
source-only comparison
dynamic dispatch unresolved
external state/time/randomness/env
runtime unavailable
unmatched path
side-effect comparison incomplete
exception envelope incomplete
```

Never let similarity or Doppelganger score become an equivalence verdict.
