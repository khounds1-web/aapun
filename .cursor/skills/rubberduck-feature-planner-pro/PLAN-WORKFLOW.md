# Plan Workflow

## Output contract

The plan produces tests and a sealed contract, not production code.

## Phase checklist

| Phase | Requirement |
|---|---|
| P0 | repo, base commit, feature description pinned |
| P1 | codegen session started or resumed |
| P2 | Fit Pack acquired and persisted |
| P3 | RubberDuck full codebase and semantic enrichment |
| P4 | Fit Pack gaps and semantic corrections |
| P5 | existing patterns discovered |
| P6 | integration point selected |
| P7 | off-limits files declared |
| P8 | reusable symbols declared |
| P9 | duplicate/doppelganger preflight |
| P10 | tests-first diff produced |
| P11 | test diff validated |
| P12 | sealed_plan.json generated |
| P13 | SIGNOFF_TEMPLATE.json generated |
| P14 | halt for signoff |

No halfway summary unless a hard blocker occurs.
